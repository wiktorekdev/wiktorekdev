## 01 · GeoHelper

Steam GeoGuessr companion.

While you play, it surfaces live coordinates, country, region, road, postcode, flag, and a map preview. Themes and full layout editing included. Connects through Chrome DevTools Protocol on Steam. No game injection.

**Stack:** Tauri, Rust, TypeScript, React  
**Links:** [website](https://geohelperapp.vercel.app/) · [latest release](https://github.com/wiktorekdev/geohelper/releases/latest) · [source](https://github.com/wiktorekdev/geohelper)

<br/>

## 02 · Typecast

Image to character art in the browser.

Drop a photo, pick a charset, export HD through 8K. Zoomable stage, color modes, multi-language UI. Processing stays on device. No accounts, no upload API.

**Stack:** React, Vite, TypeScript, Web Workers  
**Links:** [live app](https://typecast2.vercel.app) · [source](https://github.com/wiktorekdev/typecast)
