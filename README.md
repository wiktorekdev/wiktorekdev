## what I'm shipping

### GeoHelper
**Steam companion for GeoGuessr.** Live coordinates, place details, map preview, themes, full layout edit mode. Reads CDP. No injection, no DLL mess.

[Open site](https://geohelperapp.vercel.app/) · [Download](https://github.com/wiktorekdev/geohelper/releases/latest) · [Source](https://github.com/wiktorekdev/geohelper)

### Typecast
**Images become binary, ASCII, and character art.** Stage zoom, color modes, HD-8K export, multi-language UI. Runs entirely in the browser. Pixels never leave the device.

[Launch app](https://typecast2.vercel.app) · [Source](https://github.com/wiktorekdev/typecast)
